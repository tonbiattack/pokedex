# Nuxt Learning Docs

この `docs` フォルダは、この PokeDex プロジェクトを題材に `Nuxt 3` を学ぶための学習資料です。

## まず読む順番

1. [01-nuxt-overview.md](/c:/apps/pokedex/docs/01-nuxt-overview.md)
2. [02-pokedex-walkthrough.md](/c:/apps/pokedex/docs/02-pokedex-walkthrough.md)
3. [03-study-tasks.md](/c:/apps/pokedex/docs/03-study-tasks.md)

## この資料の狙い

- Nuxt の基本機能を、このリポジトリの実装と結びつけて理解する
- `pages`、`layouts`、`composables`、`server/api` の役割分担を掴む
- 単に読むだけでなく、小さく改造しながら学べるようにする

## このプロジェクトで特に学びやすい点

- `pages` ベースルーティング
- `NuxtLayout` と `NuxtPage`
- `useFetch` を使ったデータ取得
- `server/api` を使ったサーバー API
- `runtimeConfig` を使った外部 API の設定
- `composables` を使った状態とロジックの再利用
- `useSeoMeta` を使った SEO 設定
